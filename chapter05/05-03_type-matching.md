# Type Matching

Match the function to its type signature.

1. Functions:
    a) `not`
    b) `length`
    c) `concat`
    d) `head`
    e) `(<)`

2. Type signatures:
    a) `_ :: [a] -> a`
    b) `_ :: [[a]] -> [a]`
    c) `_ :: Bool -> Bool`
    d) `_ :: [a] -> Int`
    e) `_ :: Ord a => a -> a -> Bool`

Answer:
a) `not :: Bool -> Bool`
b) `length :: [a] -> Int`
c) `concat :: [[a]] -> [a]`
d) `head :: [a] -> a`
e) `(<) :: Ord a => a -> a -> Bool`
